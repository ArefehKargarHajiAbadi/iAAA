# Python Fundamentals & Data Science Basics

Welcome to my Python learning repository! 

This repository contains a structured series of Jupyter Notebooks designed to take you from the very basics of Python syntax to advanced data manipulation and visualization techniques. It is suitable for beginners and those looking to refresh their core programming skills.

##  Repository Structure

Here is an overview of the topics covered in each notebook:

| File Name | Description |
| :--- | :--- |
| **01_variables.ipynb** | **Variables & Data Types:** Understanding how to store and manage data. |
| **02_strings.ipynb** | **Text Processing:** Working with text, formatting strings, and manipulation. |
| **03_lists.ipynb** | **Lists:** Creating and managing ordered collections of items. |
| **04_tuples.ipynb** | **Tuples:** Understanding immutable (unchangeable) sequences. |
| **05_dictionary.ipynb** | **Dictionaries:** Working with key-value pairs and data mapping. |
| **06_control_statement.ipynb** | **Logic & Loops:** Using `if/else` statements, `for` loops, and `while` loops. |
| **07_functions.ipynb** | **Functions:** Writing reusable blocks of code to clean up your scripts. |
| **08_OOP.ipynb** | **Object-Oriented Programming:** Classes, Objects, Inheritance, and Encapsulation. |
| **09_numpy.ipynb** | **NumPy:** High-performance numerical computing and array manipulation. |
| **10_matplotlib.ipynb** | **Matplotlib:** Visualizing data with 2D plots and graphs. |
| **11_pandas.ipynb** | **Pandas:** Data analysis, reading CSV/Excel files, and handling DataFrames. |
| **12_files_and_error_handling.ipynb** | **I/O & Errors:** Reading/writing files and writing robust code using `try/except`. |
